# Scratch Outputs

This directory is reserved for ad hoc smoke runs, debug outputs, and temporary
exploration artifacts that should not be treated as formal benchmark results.

Tracked files:

- this `README.md`

Untracked runtime contents may be organized under family-specific subfolders.

Recommended convention:

- `experiment/artifacts/scratch/<family_id>/<label>.csv`

Anything under `scratch/` is disposable and should not be cited as a formal
benchmark result.
