# MTGNN Official-Core Wrapper

- Source: `https://github.com/nnzhan/MTGNN.git`

- Adapter owner: `experiment/families/world_model_official_baselines_v2/adapters/mtgnn_official_core.py`

This wrapper pins the official MTGNN source for v2 official-core baseline
experiments. The v2 adapter may adjust input projection, node count, calendar
conditioning, and output parameterization for the `world_model_v1` bundle, but
the graph temporal core must resolve to this source tree.
